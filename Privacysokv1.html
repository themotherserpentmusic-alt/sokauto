Privacy Policy for SOK V1

Last updated: March 21, 2026


SOK V1 is a personal automation tool that allows the owner to post video content (Reels) to their own Facebook Page and Instagram Professional account using Meta's Graph API.


Data collected and processed

- The app uses a System User access token to authenticate with the owner's Facebook Business assets.

- No personal data from third-party users is collected, stored, shared or processed.

- The only data handled is temporary OAuth/System User tokens required for API access, which are not stored long-term and are used solely for posting content to owned accounts.


Data sharing

- No data is shared with any third parties except Meta (required for API functionality).


Data retention

- Tokens are stored securely only as long as needed for automation. They can be revoked or regenerated at any time.


Your rights

- As this is a personal/internal tool with no external users, no user accounts or personal profiles exist.

- To revoke access: remove the System User token in Meta Business Settings or delete the app authorization in your Facebook account settings.
