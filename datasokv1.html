Data Deletion Instructions for SOK V1

Last updated: March 21, 2026


SOK V1 is a private, personal-use automation tool owned and operated solely by the app developer. It uses Meta's Graph API to post video content (Reels) to the developer's own Facebook Page and linked Instagram Professional account via a System User access token.


Important:

- This tool does NOT collect, store, share, or process any personal data from external users, third parties, or the general public.

- No user accounts are created.

- No profiles, emails, phone numbers, locations, or other personal information is gathered or retained.

- The only data involved is the temporary System User access token, which is used exclusively for API authentication to the developer's own assets.


Because no persistent personal data from users is collected or stored by SOK V1:

- There is no user data to delete.

- There are no end-user accounts or personal records associated with this app.


How to revoke access / end authorization:

1. Go to your Facebook account Settings → Business Integrations (or Apps and Websites).

2. Find and remove "SOK V1" (or the app name).

3. Alternatively, in Meta Business Settings → Users → System Users → Delete or regenerate the System User token.


This action immediately revokes all API access and ends any token-based functionality.
